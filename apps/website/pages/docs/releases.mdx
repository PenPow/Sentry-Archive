# Sentry Release Notes

This section detailes the changelog from our most recent releases. You can read detailed changelogs, with specific code changes over at our [releases](https://github.com/PenPow/Sentry/releases)

## Version 1.0.0

This is the big Sentry Rewrite. Here is the rundown of the main changes:

- New Architecture for better reliability and development experience
- Improved script management through [Interval](https://interval.com)
- New Audit-Log design
- Freezing Punishments
- Complete rewrite of our antivirus
- Implementation of "Phishertools", our scam domain checker
- This Website

and other changes to improve the design experience for users and developers.
