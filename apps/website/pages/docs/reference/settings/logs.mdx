# Logging Settings

Adjust guild related settings for audit logs. This option should not be changed often, as it can occasionally cause compatibility errors when working with really old punishments.

## Syntax

import { Callout } from 'nextra-theme-docs'

<Callout type="info" emoji="ℹ️">
  `/settings logs [audit-channel]`
</Callout>

### Options

| Option        | Type           | Required | Description                                           |
|---------------|----------------|----------|-------------------------------------------------------|
| Audit Channel | (Text) Channel | Yes      | What channel Sentry should set for its audit logging. |

## Permissions

This command requires the `Administrator` permission.
