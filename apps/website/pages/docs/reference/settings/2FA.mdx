# 2FA Settings

Adjust guild related settings for 2FA.

## Syntax

import { Callout } from 'nextra-theme-docs'

<Callout type="info" emoji="ℹ️">
  `/settings 2fa [enforce-for-moderation]`
</Callout>

### Options

| Option                 | Type    | Required | Description                                                                       |
|------------------------|---------|----------|-----------------------------------------------------------------------------------|
| Enforce For Moderation | Boolean | Yes      | Whether Sentry should require that moderators have 2FA enabled on their accounts. |

## Permissions

This command requires the `Administrator` permission.
