# Punishment Settings

Here you can adjust settings related to punishments for your guild.

## Syntax

import { Callout } from 'nextra-theme-docs'

<Callout type="info" emoji="ℹ️">
  `/settings punishments [auto-freeze]`
</Callout>

### Options

| Option      | Type    | Required | Description                                                           |
|-------------|---------|----------|-----------------------------------------------------------------------|
| Auto Freeze | Boolean | Yes      | Whether Sentry should automatically freeze punishments after creation |

## Permissions

This command requires the `Administrator` permission.
