# Setup 2FA

If you want to improve your account security, setup an extra layer of authentication through Sentry.

By adding 2FA to your account, you can improve your security by requiring that a code from your authenticator app is inputted before making moderation actions, or modifying settings.

Your 2FA settings are global throughout the entirety of Sentry.

## Syntax

import { Callout } from 'nextra-theme-docs'

<Callout type="info" emoji="ℹ️">
  `/2fa setup`
</Callout>

## Permissions

This command doesnt require any specific permissions to run.
